# Llaves (Keys)

- Primary Key
- Super Key
- Candidate Key
- Foreing Key
- Composite Key
- And more...

## Primary Key

- Identifica un registro de forma unica
- Una tabla puede tener varias primary keys
- La llave primaria esta basada en los requerimientos

## Candidate Key

- Un atributo o conjunto de ellos que identifican de forma unica.
- Menos la llave primaria, los demas se consideran candidate keys.

## Super Key

- Es un conjunto de atributos que puede identificar de forma unica.
- Es un superconjunto de una clave candidata.

## Foreign Key

- Foreign Keys (llaves foraneas) son usadas para apuntar a la llave primaria de otra tabla.

## Composite Key

- Cuando una clave primaria consta de de mas de un atributo, se conoce como clave compuesta (Composite Key)
